# Landing Page – Lash Designer
Landing page desenvolvida para apresentar os serviços de uma profissional especializada em extensão de cílios, com foco em transmitir confiança, explicar os benefícios do procedimento e facilitar o contato com novas clientes.

O site foi pensado para guiar o visitante desde o primeiro contato com a profissional até o momento de decisão, apresentando informações claras sobre os serviços oferecidos.

## Objetivo do Projeto
O objetivo desta landing page é:

- Apresentar a profissional e sua especialidade

- Explicar por que investir em extensão de cílios

- Mostrar os serviços disponíveis

- Aumentar a confiança da cliente antes do agendamento

- Facilitar o contato e conversão de novas clientes
  

A página foi estruturada de forma estratégica para que a visitante:

1. Conheça a profissional

2. Entenda os benefícios da extensão de cílios

3. Visualize os serviços oferecidos

4. Se sinta segura para entrar em contato

## Processo de Design
Antes do desenvolvimento, todo o projeto foi planejado no Figma, onde foram criados:

- Wireframe da página
- Design visual da landing page
- Prototipagem interativa
  
A prototipagem permitiu que a cliente visualizasse como o site funcionaria antes mesmo do desenvolvimento, possibilitando ajustes e validações antecipadas.
Isso garantiu que o resultado final estivesse alinhado com a expectativa da cliente.

## Tecnologias Utilizadas
- HTML5 – Estrutura da página
- CSS3 – Estilização e layout responsivo
- JavaScript – Interatividade e comportamento da página
- Figma – Design da interface e prototipagem

## Funcionalidades da Landing Page
- Seção de apresentação da profissional
- Explicação dos benefícios da extensão de cílios
- Apresentação dos serviços disponíveis
- Estrutura pensada para conversão de clientes
- Layout responsivo
- Navegação clara e intuitiva
